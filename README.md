## Sobre o Repositório

Este espaço guarda minha evolução no desenvolvimento de aplicativos nativos para Android. O foco principal é a criação de interfaces modernas e reativas utilizando **Jetpack Compose**, aliado à consolidação dos fundamentos da linguagem **Kotlin**. 

Aqui você encontrará desde os primeiros passos com a sintaxe do Kotlin até a construção de layouts e aplicativos funcionais criados no Android Studio.

### Principais Tópicos Abordados
* **Fundamentos de Kotlin:** Sintaxe básica, variáveis, funções, orientação a objetos e null safety.
* **Jetpack Compose:** Construção de UIs declarativas, uso de Modifiers, gerenciamento de Estado (State) e navegação.
* **Ecossistema Android:** Uso do Android Studio, configuração do Gradle e execução no emulador (AVD).
